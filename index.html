The Problem

Due diligence takes weeks.
Deals move in days.

01 — TIME

4–8 weeks per deal

Traditional M&A due diligence requires 5–15 analysts and takes weeks — by which time the opportunity may be gone.

02 — COST

$50K–$500K in fees

Advisory firms charge a premium for the research, analysis, and memo writing that AI can now do in minutes.

03 — SIGNAL/NOISE

80% gathering, 20% thinking

Analysts spend most of their time aggregating data — not on the judgment calls that actually matter for the deal.

How It Works

From company name
to investment decision.

1

Enter a company name and context

Run python main.py or open the dashboard. Type the target company and any context you have — funding stage, sector, recent news.

2

5 agents coordinate through Band

Each agent authenticates with its own band_a_... key. They post to a shared chat room and @mention the next agent — no orchestration server needed.

3

Watch the pipeline in Band's UI

Open the chat room in app.band.ai and watch the agents appear in real time, each @mentioning the next. Full audit trail by default.

4

Human reviewer approves the memo

@HumanReview presents the final investment memo. The human decides: APPROVE, REJECT, or REQUEST CHANGES. No memo ships without a human decision.

Why Band?

Multi-agent coordination through conversation.

  • Each agent authenticates with its own Agent API key — no shared credentials
  • Agents @mention the next — no message broker, no queue, no orchestration server needed
  • The full pipeline is visible in the Band chat UI — transparent and auditable by design
  • Sibling agents can be added as participants in any chat room without a contact relationship
